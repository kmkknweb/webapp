✍️ ลงทะเบียนรับสิทธิ์

กรอกข้อมูลเพื่อยืนยันตัวตนและรับสิทธิ์ร่วมกิจกรรม

หมายเหตุ
• 1 เบอร์โทรควรลงทะเบียน 1 ครั้ง หากพบข้อมูลซ้ำ คณะกรรมการสามารถพิจารณาตัดสิทธิ์ได้
• ระบบจะบันทึกว่าคุณมาจากคำแนะนำของใคร และบันทึกวันที่ติดตั้งแอปด้วย
• ลิงก์แนะนำจะพาเพื่อนมายังหน้านี้พร้อมรหัสของคุณโดยอัตโนมัติ
📲 หากท่านยังไม่เคยติดตั้ง application นี้มาก่อน ให้ติดตั้งแอปเพื่อการรับสิทธิ์ชิงรางวัลจะได้สมบูรณ์
✅ ติดตั้งแอปเรียบร้อยแล้ว ขอบคุณที่ร่วมกิจกรรม!
🌐 กดเพื่อติดตั้ง app

สำหรับผู้ที่ติดตั้งไม่เป็น กดดูขั้นตอนแบบภาพประกอบได้ที่นี่

📲 ติดตั้งแอป / เพิ่มไปหน้าจอหลัก
⋮ เมนู Chrome มุมขวาบน
🏠 เปิดจากหน้าจอมือถือ
1
เปิดลิงก์ด้วย Chrome
ถ้าเปิดจาก LINE แล้วติดตั้งไม่ได้ ให้กดคัดลอกลิงก์จากหน้า install แล้วนำไปวางใน Chrome
2
กดปุ่มติดตั้ง ถ้ามีขึ้น
ถ้า Chrome แสดงปุ่ม “ติดตั้งแอป” ให้กดปุ่มนั้นได้ทันที
⋮
ถ้าปุ่มไม่ขึ้น ให้ใช้เมนู Chrome
กดเมนู ⋮ มุมขวาบน แล้วเลือก เพิ่มไปยังหน้าจอหลัก
✓
เสร็จแล้วเปิดจากไอคอนบนหน้าจอ
เมื่อติดตั้งสำเร็จ จะมีไอคอนอากงอาม่าบนหน้าจอมือถือ ใช้งานได้เหมือนแอป
Safari
🔗 ปุ่มแชร์
➕ เพิ่มไปยังหน้าจอโฮม
1
คัดลอกลิงก์ แล้วเปิด Safari
iPhone ต้องเปิดเว็บด้วย Safari ก่อน จึงจะเพิ่มเป็นไอคอนบนหน้าจอโฮมได้
2
วางลิงก์ในช่องเว็บด้านบน
วางลิงก์ install แล้วกดเข้าเว็บ รอให้หน้าเว็บโหลดเสร็จ
↗
กดปุ่มแชร์ของ Safari
ปุ่มแชร์เป็นรูปสี่เหลี่ยมมีลูกศรชี้ขึ้น อยู่ด้านล่างของ Safari
＋
เลือก “เพิ่มไปยังหน้าจอโฮม”
เลื่อนเมนูลงมา แล้วเลือก เพิ่มไปยังหน้าจอโฮม จากนั้นกด เพิ่ม
↓ หลังติดตั้งแล้ว ให้กลับมาหน้านี้เพื่อลงทะเบียนร่วมกิจกรรม หรือเปิดจากไอคอนบนหน้าจอมือถือได้เลย

iPhone / iPad — ทำตามขั้นตอนนี้

  1. เปิดลิงก์นี้ด้วย Safari
  2. กดปุ่ม Share (กรอบมีลูกศรชี้ขึ้น)
  3. เลื่อนลงแล้วกด Add to Home Screen
  4. กด Add มุมขวาบน

ถ้าปุ่มติดตั้งไม่ขึ้น ให้กดเมนู ⋮ ของ Chrome แล้วเลือก เพิ่มไปหน้าจอหลัก
