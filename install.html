กำลังตรวจสอบอุปกรณ์...

ระบบจะแนะนำวิธีติดตั้งที่เหมาะกับมือถือของคุณ

📲 เปิดใน Chrome เพื่อติดตั้ง เข้าใช้งานก่อน
ขอพร ดูดวง ทำบุญ กิจกรรม

Android / Chrome

ถ้าปุ่ม ติดตั้งแอป ขึ้น ให้กดติดตั้งได้ทันที

ถ้าปุ่มไม่ขึ้น ให้กดเมนู ⋮ มุมขวาบนของ Chrome แล้วเลือก เพิ่มไปยังหน้าจอหลัก

Android / LINE

ตอนนี้คุณเปิดจาก LINE ซึ่งมักไม่รองรับปุ่มติดตั้ง PWA โดยตรง

  1. กดปุ่ม เปิดใน Chrome เพื่อติดตั้ง
  2. เมื่อเปิดใน Chrome แล้ว ให้กด ติดตั้งแอป
  3. ถ้าปุ่มไม่ขึ้น ให้กดเมนู ⋮ แล้วเลือก เพิ่มไปยังหน้าจอหลัก

iPhone / iPad

iPhone จะไม่มีปุ่มติดตั้งแบบ Android ต้องเพิ่มลงหน้าจอโฮมผ่าน Safari

  1. เปิดหน้านี้ด้วย Safari
  2. กดปุ่ม แชร์ ด้านล่างของ Safari
  3. เลือก เพิ่มไปยังหน้าจอโฮม
  4. กด เพิ่ม

ถ้าเปิดจาก LINE ให้กดเมนูของ LINE แล้วเลือกเปิดใน Safari หรือคัดลอกลิงก์ไปเปิดใน Safari

คำแนะนำ

หากต้องการติดตั้งเป็นไอคอนบนหน้าจอหลัก แนะนำให้เปิดหน้านี้ด้วย Chrome บน Android หรือ Safari บน iPhone

ติดตั้งแล้วได้อะไร

เปิดใช้งานได้สะดวกจากหน้าจอมือถือ เหมือนแอปจริง เหมาะสำหรับติดตามกิจกรรม ขอพร ดูดวง และร่วมทำบุญออนไลน์
