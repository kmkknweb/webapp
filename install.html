หน้าแนะนำการติดตั้ง

ถ้าเครื่องรองรับ จะมีปุ่มติดตั้งแอปแสดงขึ้นอัตโนมัติ

เข้าใช้งานก่อน
ขอพร ดูดวง ทำบุญ กิจกรรม

ถ้าปุ่มติดตั้งไม่ขึ้น

ให้คัดลอกลิงก์ด้านล่าง แล้วนำไปเปิดเองใน Chrome บน Android หรือ Safari บน iPhone/iPad

คัดลอกแล้วเปิด Chrome/Safari → วางลิงก์ → หากรองรับจะมีปุ่มติดตั้ง หรือใช้เมนู “เพิ่มไปยังหน้าจอหลัก”

วิธีเพิ่มลงหน้าจอหลัก

  1. Android / Chrome: ถ้าปุ่มติดตั้งไม่ขึ้น ให้กดเมนู ⋮ แล้วเลือก เพิ่มไปยังหน้าจอหลัก
  2. iPhone / iPad: เปิดด้วย Safari → กดปุ่มแชร์ → เลือก เพิ่มไปยังหน้าจอโฮม

ติดตั้งแล้วได้อะไร

เปิดใช้งานได้สะดวกจากหน้าจอมือถือ เหมือนแอปจริง เหมาะสำหรับติดตามกิจกรรม ขอพร ดูดวง และร่วมทำบุญออนไลน์
