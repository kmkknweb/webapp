ติดตั้งแอปอากงอาม่า

หน้านี้เป็นหน้าช่วยติดตั้งสำหรับผู้ที่เปิดจาก LINE หรือเบราว์เซอร์ในแอป ให้เปิดด้วย Chrome หรือ Safari ก่อนเพิ่มแอปไว้ที่หน้าจอหลัก

📲 เปิดใน Chrome เพื่อติดตั้ง เปิดแอป
หากกดจาก LINE แล้วติดตั้งไม่ได้ ให้กดเปิดด้วย Chrome หรือ Safari ก่อน
Android / Chrome

วิธีติดตั้งบน Android

  1. กดปุ่ม เปิดใน Chrome เพื่อติดตั้ง
  2. เมื่อเปิดใน Chrome แล้ว กดเมนู ⋮
  3. เลือก ติดตั้งแอป หรือ เพิ่มไปหน้าจอหลัก
  4. กลับไปเปิดจากไอคอน “อากงอาม่า” บนหน้าจอมือถือ

ถ้าปุ่มติดตั้งยังไม่ขึ้น ให้ลองรีเฟรชหน้าเว็บใน Chrome อีกครั้ง หรือเปิดหน้าแรกของแอปก่อน

iPhone / iPad

วิธีเพิ่มลงหน้าจอโฮม

  1. ถ้าเปิดจาก LINE ให้กดเมนูของ LINE แล้วเลือก เปิดใน Safari
  2. เมื่ออยู่ใน Safari ให้กดปุ่ม แชร์ ด้านล่าง
  3. เลือก Add to Home Screen หรือ เพิ่มไปยังหน้าจอโฮม
  4. กด Add / เพิ่ม

iPhone จะไม่มีปุ่มติดตั้งแบบ Android ต้องเพิ่มผ่าน Safari เท่านั้น

หมายเหตุ

หน้า install นี้เป็นเพียงหน้าคู่มือและทางลัดสำหรับเปิดเบราว์เซอร์ที่ถูกต้อง ส่วนตัวแอปจริงอยู่ที่ index.html
